# Wildlife Rescue Coordination Platform

This document outlines the setup and current status of the project being developed with the Gemini CLI.

## Project Setup Status:

*   **Backend:** Spring Boot (Maven) with Java 17, Web, WebSocket, and Lombok dependencies configured.
*   **Frontend:** ReactJS application scaffolded (user will run `npx create-react-app .` in the `frontend` directory).

## Next Steps:

2.  Implement core features for both frontend and backend based on the agreed-upon plan.
3.  Integrate frontend and backend communication.
